# مساحتي ☕
### تطبيق حجز مساحات العمل في الكافيهات

---

## هيكل المشروع

```
masakti/
├── App.js                      ← نقطة الدخول + التنقل
├── package.json                ← المكتبات المطلوبة
└── src/
    ├── theme/
    │   └── index.js            ← الألوان، المسافات، الخطوط
    └── screens/
        ├── HomeScreen.js       ← الشاشة الرئيسية (قائمة الكافيهات)
        └── BookingScreen.js    ← شاشة الحجز التفصيلية
```

---

## تشغيل المشروع

### المتطلبات
- Node.js 18+
- npm أو yarn
- Expo Go على جوالك (iOS أو Android)

### خطوات التشغيل

```bash
# 1. ثبّت المكتبات
npm install

# 2. شغّل التطبيق
npx expo start

# 3. افتح Expo Go على جوالك وسكان الـ QR
```

---

## المميزات الحالية

- **الشاشة الرئيسية**: قائمة الكافيهات مع خريطة المقاعد الحية
- **الفلاتر**: تصفية حسب الإتاحة، الغرف الخاصة، المسافة، مستوى الهدوء
- **البحث**: بحث بالاسم أو الحي
- **إحصائيات لحظية**: عدد الأماكن المتاحة في الوقت الفعلي
- **شاشة الحجز**: اختيار الوقت، المدة، نوع المقعد
- **ملخص الحجز**: تأكيد بصري قبل الإتمام

---

## التصميم

- **النمط**: فاخر داكن (Dark Luxury)
- **اللون الرئيسي**: Lime Electric `#C8F56A`
- **الخلفية**: `#0A0A0F` (أسود عميق)
- **الكروت**: `#181824` مع حدود شفافة خفيفة

---

## الخطوات القادمة (مقترحة)

1. **Backend**: ربط بـ Firebase أو Supabase للبيانات الحية
2. **الخريطة**: دمج Google Maps لعرض الكافيهات على الخريطة
3. **الإشعارات**: Expo Notifications للتذكير قبل الموعد
4. **الدفع**: دعم Apple Pay / STC Pay للحجوزات المدفوعة
5. **لوحة الكافيه**: واجهة للكافيه لتحديث المقاعد المتاحة
